INSTALLTION\
LearnBygame is written in Python,so you'll need a Python interpreter to get anything working.It also depends on several Python modules;see the DEPENDENCISE section below if you have any problems.if you do have all  dependencies,you should be able to run the current demo just by running the main LearnByGame script with a Python interpreter.From a shell,the folling command should do the trick:\
\
$ python LearnByGame.py\
\
Users of graphical file managers can probably  double-click the LearnByGame script to achieve the same effect.\
\
